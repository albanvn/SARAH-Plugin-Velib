Le plugin 'Velib' permet de connaître les vélos disponible ou les places libres disponibles aux stations Velib

Pour le moment, ce plugin fonctionne (contrats JCDecaux) dans les villes suivantes:
  • Amiens
  • Besancon
  • Bruxelles-Capitale
  • Cergy-Pontoise
  • Creteil
  • Kazan
  • Lillestrom
  • Ljubljana
  • Lyon
  • Luxembourg
  • Goteborg
  • Marseille
  • Mulhouse
  • Namur
  • Nantes
  • Nancy
  • Paris
  • Rouen
  • Toulouse
  • Santander
  • Seville
  • Stockholm
  • Toyama
  • Valence
  • Vilnius


Configuration et paramétrages du plugin.

Pour que le plugin fonctionne correctement, il faut renseigner dans la configuration du plugin, le champs 'city' et le champs 'stations'.
Pour le champs 'city', le champs doit contenir une des villes décrites ci-dessus. Pour le champs 'stations', rendez-vous sur le site Vélib ou équivalent de votre ville (pour paris: http://www.velib.paris/Plan-stations) et repérer les numéros de stations proches de vous, puis rentrer ces numéros sous forme d'une liste séparés dont les stations sont séparés par une virgule.
Le champs 'apiKey' est facultatif (le plug in utilise mon compte JCDecaux) : si vous voulez créer votre propre ID apiKey JCDecaux, rendez-vous sur le site JCDecaux développeur et créer un compte puis demandez une apiKey. Vous renseignerez alors cet apiKey dans le champs 'apiKey' du plugin Velib.


Commandes reconnues par SARAH.

  • Est-ce que il y a des vélib disponibles: Sarah vous indiquera les vélib disponibles autour de vous
  • Est-ce que il y a des places de vélib disponibles: Sarah vous indiquera les places libres disponibles autour de vous
